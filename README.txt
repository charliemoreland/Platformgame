# Platform Game

## Operating Instructions
Run the file to open up a window to dodge enemies and collect all the coins.

## Game Play
Use left and right arrows to move and space to jump. Must collect all 21 coins and reach the green exit tube to win. 

##
There are 7 platform 2 of which move(One vertically and one horizontally).
there are 21 coins.
There are 2 enemies that pace left and right.
